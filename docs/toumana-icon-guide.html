Design Principles

The Toumana icon system follows Apple and Airbnb design principles, creating a clean, warm, and welcoming visual language that reflects Mediterranean hospitality.

Why Lucide Icons?

Characteristic Apple/Airbnb Standard Lucide Implementation
Stroke Weight 1.5-2px consistent 2px default
Stroke Caps Rounded, friendly Round linecap/linejoin
Style Outlined default Outlined with fill option
Geometry Clean, minimal Simple, recognizable
Warmth Human, approachable Soft rounded corners

Size Scale

Five sizes aligned with the Toumana spacing system. Default size is 20px (md) for most UI elements.
